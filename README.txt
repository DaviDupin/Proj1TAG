Para compilar é necessário inserir o comando no terminal:
g++ -std=c++11 main.cpp parser.cpp -o arquivo

--------------------------------------------------

Em seguida para executar, basta inserir:
./arquivo

--------------------------------------------------

A tela inicial mostra um menu com as opções implementadas.
Para escolher a função a ser executada, basta inserir
um inteiro correspondente à opção listada.

Em seguida o programa processa as informações do arquivo
karate.gml e imprime na tela o resultado da operação pedida.
